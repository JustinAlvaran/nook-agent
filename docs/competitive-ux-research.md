# Competitive UX research

Research accessed 2026-07-12.

- ChatGPT agent emphasizes visible control and confirmation around consequential actions. Nook adopts exact approval tickets and durable receipts, while narrowing the MVP to local tools. Source: https://help.openai.com/en/articles/11752874-chatgpt-agent
- Microsoft Copilot Vision makes shared visual context explicit rather than implying unrestricted screen access. Nook's future desktop/browser capability must be an explicit revocable grant. Source: https://support.microsoft.com/en-US/microsoft-copilot/using-copilot-vision-with-microsoft-copilot
- Replika makes avatar personalization persistent and user-owned. Nook loads account state before a device draft and treats appearance separately from capability. Source: https://help.replika.com/hc/en-us/articles/360046050652-How-do-I-change-my-Replika-s-avatar
- Finch separates shops and collectible customization from core care. Nook similarly separates owned/equipped cosmetics from agent permissions. Source: https://help.finchcare.com/hc/en-us/articles/37935977276813-Shops-in-Finch-Outfits-Travel-and-More

Differentiation: the Nook room is not a chat skin. It exposes plan, exact tool/version/input, approval, execution state, verification, memory consent, and receipt as first-class product objects. The mascot's motion is downstream of durable state rather than decorative fake activity.
