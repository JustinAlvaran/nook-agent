# Research security

Retrieved content is untrusted data. It cannot modify instructions, grant permission, register tools, select arbitrary capabilities, access secrets, trigger actions, or write memory.

Only HTTPS URLs are accepted. Localhost, `.local`, loopback, link-local, and RFC1918 IPv4 destinations are rejected. Allow/blocked domain lists are enforced with exact or subdomain matching. Results are bounded to ten sources and snippets to 1,000 characters. Full pages are not stored.

Current claims must retain citations. Missing publication dates are shown as unknown rather than fabricated.
